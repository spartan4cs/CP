   The "Macros" folder contains useful keyboard macro commands.

   The "Colors" folder contains custom colors schemes.

   The "Shell" folder contains  some  addons  that  could  be  used  for
integrating FAR with the Windows shell.

   The "SetUp" folder contains quick setup options.

   The "XLat" folder contains Xlat-tables for transliteration.

   To install any of the provided .farconfig files just run:
   Far.exe /import filename.farconfig

   To install any of the provided .lua files just copy them to
%FARPROFILE%\Macros\scripts directory (or any of its subdirectories).
   
   All included .farconfig files contain descriptions. Switch to "Long
descriptions" view mode in order to read them.
